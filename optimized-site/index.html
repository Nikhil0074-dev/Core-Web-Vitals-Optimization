7 Optimization Techniques

Every technique was measured before and after with Lighthouse to confirm impact.

[IMG]

Image Optimization

Converted PNG/JPEG to WebP & AVIF using Squoosh. Added explicit width/height to every . Used for format negotiation. Result: −70% image payload.

>>

Eliminate Render-Blocking JS

Removed jQuery entirely (87 KB). Applied defer to all scripts. Extracted unused functions with tree-shaking. Result: FID improved from 250 ms -> 60 ms.

[CDN]

CDN & Edge Caching

Deployed assets via Cloudflare (300+ PoPs). Enabled Brotli compression, HTTP/3, and aggressive cache-control headers. Result: TTFB dropped 78%.

[CSS]

CSS Critical Path

Inlined 2 KB of above-fold CSS directly in . Loaded remaining 14 KB asynchronously. Ran PurgeCSS to eliminate 2,400 bytes of unused rules.

[REP]

Lazy Loading

Applied loading="lazy" to all below-fold images. Used fetchpriority="high" only on the LCP candidate. Intersection Observer defers non-critical third-party widgets.

[PKG]

Code Splitting

Replaced monolithic 280 KB bundle with route-based chunks via dynamic import(). Only the current page's JS loads on first visit. Result: −86% first-load JS.

[SW]

Service Worker Cache

Pre-caches app shell on install. Uses stale-while-revalidate for images and network-first for HTML. Provides offline fallback. Repeat visits load in < 200 ms.
